# Animal Handbook

Welcome to the Animal Handbook, a place where you can learn everything about animals!

## Overview

it makes it possible to read descriptions of animals from all over the world. In other words, it is an encyclopedia of animals.

## Features

- **Convenient Search:** Search for animals by name
- **Favourites animals:** Adding animals to favorites
- **Smooth Transitions:** Enjoy smooth animations between screen transitions.


## How to Use

1. Launch the app.
2. First you will see a list of animal categories. Choose the one that interests you.
3. Then you will see a list of animals in this category.
4. Then just choose any animal and learn something new.
5. If you want to save someone, just add to favorites.
